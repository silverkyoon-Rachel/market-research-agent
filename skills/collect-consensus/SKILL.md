# 마켓 컨센서스 종합 스킬

이 스킬은 다른 수집 스킬(collect-stock, collect-financials, collect-news, collect-analyst)의 결과를 종합하여 마켓 컨센서스 리포트를 생성합니다.

## 이 스킬의 역할

개별 수집 스킬이 데이터를 모은 후, 이 스킬이 모든 데이터를 하나의 종합 리포트로 통합합니다.

## 종합 분석 항목

### 1. 밸류에이션 분석
수집된 데이터를 기반으로:
- 현재 PER/PBR vs 업종 평균 비교
- 목표가 대비 현재가 괴리율
- 시가총액 적정성 (재무 대비)

### 2. 성장성 분석
재무제표 데이터에서:
- 매출 성장률 (YoY)
- 영업이익 성장률 (YoY)
- 마진 개선/악화 추세

### 3. 시장 심리 분석
뉴스 + 애널리스트 데이터에서:
- 뉴스 감성 트렌드 (최근 긍정/부정 비율)
- 애널리스트 의견 트렌드 (상향/하향 조정)
- M&A 시그널 종합

### 4. 리스크 요인
모든 데이터를 종합하여:
- 재무 리스크 (부채비율, 이익 변동성)
- 시장 리스크 (주가 변동성, 외국인 매도)
- 뉴스 리스크 (부정적 보도 동향)

### 5. M&A 매력도 평가
이 프로젝트의 핵심 목적인 M&A 관점에서:
- 인수 매력도 점수 (1-10)
- 핵심 매력 포인트
- 인수 시 리스크 요인
- 적정 인수 가격 범위 (추정)

## 분석 방법

### API 키 조회 순서

API 키는 다음 순서로 확인합니다:
1. `backend/.env` 파일 (프로젝트 환경변수) — Read 도구로 파일을 읽어 해당 키 값을 확인
2. `.claude/market-research-agent.local.md` (플러그인 설정) — 1번에 없을 경우 확인

### Anthropic API 키가 있는 경우
- 수집된 모든 데이터를 Claude API에 전달
- 구조화된 프롬프트로 종합 분석 요청
- M&A 관점의 전문적 분석 생성

### Anthropic API 키가 없는 경우
- 현재 대화의 Claude 모델이 직접 분석
- 수집된 데이터의 수치적 통계 기반 분석
- 정량적 지표 중심으로 컨센서스 도출

```
| 항목 | 전년 | 금년 | 성장률 |

- 뉴스 감성: {긍정적/중립/부정적} (점수: +0.0)

```
